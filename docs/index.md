# ReconX Security Automation Platform

Welcome to the official documentation for **ReconX**, a modern reconnaissance orchestration platform. ReconX simplifies the execution of complex security workflows, centralizes vulnerability findings, and provides deep insights through an intelligent correlation engine.

## Getting Started

If you're new to ReconX, check out our [Getting Started Guide](getting-started/installation.md) to set up your environment, configure your first project, and run your initial workflow within minutes.
