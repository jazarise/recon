# Changelog

## [30.0.0-enterprise] - 2026-06-03
### Added
- **Asset Database**: SQLite/SQLAlchemy persistent storage.
- **Correlation Engine**: Automated Asset Graph building.
- **Project Workspaces**: Isolated databases and output per client.
- **Scan Profiles**: CLI support for predefined profiles.
- **Workflow Builder**: Direct CLI execution of custom YAML workflows.
- **Auto-Registration**: Dynamic plugin loading.
- **Central Config**: `config.yaml` added.
- **Structured Logging**: `core/logger.py` JSON logging.
- **Documentation Suite**: 8 markdown manuals added.
