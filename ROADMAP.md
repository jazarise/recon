# ReconX Roadmap

## Current (v1.0.x)
- Stabilize core Workflow Engine.
- Expand documentation and tutorials.
- Increase coverage of default plugins (Subfinder, Naabu, Httpx, Nuclei).

## Planned (v2.x)
- **Cloud Reconnaissance**: Native plugins for AWS, GCP, Azure enumeration.
- **Distributed Scanning**: Worker nodes capable of pulling tasks from a central queue (RabbitMQ/Redis) instead of executing locally.
- **AI-Assisted Analysis**: LLM integration to automatically score and summarize findings.
- **Frontend Dashboard**: A React/Vue.js based web interface replacing the CLI as the primary interaction method.
- **Plugin Marketplace**: A central repository for sharing and discovering plugins.
