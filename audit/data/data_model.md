# Data Model

The Unified Data Layer in ReconX defines the following canonical objects:

- **Target**: Represents the initial scope.
- **Host**: An active IP or resolved domain.
- **Subdomain**: A discovered subdomain.
- **Port**: An open network port.
- **Service**: An application running on a port.
- **Technology**: Software stack components.
- **Vulnerability**: Identified security flaws.
- **Finding**: The universal wrapper for all discovered items.
- **Scan**: Represents an execution session.
