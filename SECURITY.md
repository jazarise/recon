# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 3.x     | :white_check_mark: |
| 2.x     | :x:                |

## Reporting a Vulnerability
Email security@reconx.local.

## Triage Process & Response Timelines
| Severity | Response |
| -------- | -------- |
| Critical | 24h      |
| High     | 72h      |
| Medium   | 7d       |
| Low      | 30d      |
