# Contributing to ReconX

1. **Branch Strategy:** Fork the repo, create a feature branch (`feature/your-feature`), and submit a PR to `main`.
2. **Code Standards:** All Python code must be typed (`mypy`) and formatted with `black`.
3. **Testing:** New plugins MUST include a unit test in `tests/plugins/`.
4. **Review Process:** 1 Core Maintainer approval is required.
